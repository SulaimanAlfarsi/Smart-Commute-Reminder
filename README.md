# Smart Commute Reminder

This project is set up as a Java 17 Maven application.

## Requirements

- Java 17

## Build

Use the Maven wrapper:

```powershell
.\mvnw.cmd test
```

## Run

```powershell
.\mvnw.cmd -q exec:java
```

## Project layout

- `src/main/java` for application code
- `src/main/resources` for runtime resources
- `src/test/java` for tests
- `src/test/resources` for test resources
